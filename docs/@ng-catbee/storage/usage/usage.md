# Usage Examples

Comprehensive examples for using the @ng-catbee/storage package.

- [Basic Usage](./basic-usage) - Get, set, delete operations
- [Type-Safe Methods](./type-safe-methods) - JSON, boolean, number, enum, array methods
- [Reactive Features](./reactive-features) - Watch storage changes with observables

## Common Use Cases

### User Preferences

Store and retrieve user settings like theme, language, and layout preferences.

### Authentication Tokens

Manage access tokens and refresh tokens with automatic expiration.

### Form Data Persistence

Auto-save form data to prevent data loss on page refresh.

### Shopping Cart

Persist cart items across sessions.

### Feature Flags

Store client-side feature toggles and A/B test variants.

### Analytics Data

Buffer analytics events before sending to server.

### Cache Management

Store API responses to reduce server requests.

### Cross-Tab Communication

Synchronize state across multiple browser tabs.
